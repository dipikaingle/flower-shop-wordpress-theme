## 📌 Overview
Flower Shop is a custom WordPress theme designed for florists, flower stores, gift shops, and small eCommerce businesses.

The theme focuses on elegant design, product showcase, and smooth user experience for online shopping.

Ideal for:
- Flower shops
- Gift stores
- Small eCommerce businesses
- Boutique stores

---

## ✨ Features
- Fully custom WordPress theme (no page builder dependency)
- Elegant and modern UI design
- Fully responsive layout (mobile, tablet, desktop)
- Product showcase and shop layout
- WooCommerce ready (if integrated)
- SEO-friendly structure
- Fast loading and optimized performance
- Cross-browser compatibility

## ⚙️ Installation
1. Download or clone the flower-shop-pro  
2. Upload the theme folder to:  
   `/wp-content/themes/`  
3. Go to WordPress Admin → Appearance → Themes  
4. Activate the theme  
5. Install WooCommerce plugin (if using shop features)
